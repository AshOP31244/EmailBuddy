info Installation Instructions

computer Chrome / Edge (Desktop)
  1. Click the install icon (⊕) in the address bar
  2. Or click the three dots menu → Install Email Buddy
  3. Click Install in the popup
  4. Email Buddy will open as a standalone app!
phone_iphone Safari (iPhone/iPad)
  1. Tap the Share button (square with arrow)
  2. Scroll down and tap Add to Home Screen
  3. Tap Add in the top right
  4. Email Buddy icon will appear on your home screen!
smartphone Chrome (Android)
  1. Tap the three dots menu (⋮)
  2. Tap Add to Home screen
  3. Tap Install or Add
  4. Email Buddy will be available on your home screen!

Once installed, Email Buddy will work just like a native app!
